Project 1: Laser Game

The Laser Game is an interactive coding project built using Processing. It involves designing and programming a game where players must hit targets using simulated laser beams. The game showcases logic programming, graphics manipulation, and user interaction within the Processing environment.

Features:

  • Real-time laser movement and target detection
  • Dynamic score tracking
  • Visually engaging animations and effects
